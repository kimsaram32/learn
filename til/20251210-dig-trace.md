# 20251210 - `dig +trace` option (short)

`dig` command-line tool의 `+trace` 옵션은 DNS resolution의 tracing을 활성화한다.
최종 record를 얻기까지의 과정을 전부 나열하여 보여주며, DNS의 resolution flow를 보여준다고 할 수 있을 것이다.

이는 일반적인 DNS resolver가 수행하는 기본적인 일을 실행함으로써 구현한다.
옵션이 활성화된 상태의 dig는 root name server부터 시작하여 iterative query를 진행한다.

기본적으로 DNS는 host 입장에선 DNS resolver에 요청을 보내고 DNS 시스템의 디테일에는 신경쓰지 않는 형태인데,
이것에서 벗어나서 직접 query를 진행하는 것이라고 해석할 수 있다.
