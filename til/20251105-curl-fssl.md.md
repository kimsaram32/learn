#  20251105 - `curl -fsSL` (short)

fsSL은 curl에서 흔히 쓰이는 옵션들의 조합이다.

curl은 기본적으론 human-readable하게 동작한다.
이를 shell scripting에 쓰기 적절한 형태로 만들기 위해 붙이는 옵션들이다.

- `-f`: fail 시 적절한 error code와 함께 종료한다.
  일반적으론 실패 response를 readable 하게 반환하나 그렇지 않도록 한다.
  
- `-s`: silent mode로 동작한다.
  scripting 상에선 중간 로그가 존재하면 안 되므로 사용한다.
  
- `-S`: show error:
  silent mode여도 error는 보여주도록 한다.
  
- `-L`: 3xx redirection을 핸들링한다.
