# 20251106 - VFS in Linux (short)

VFS는 리눅스에서 사용자에게 제공하는 가상 파일 시스템이다.

- underlying file system (physical, pesudo...) 들을
  하나의 통합된 파일 시스템 형태로 유저에게 제공한다.

- underlying file system들은 VFS에 대응되는 operation을 VFS에게 제공한다.
  이를 통해 VFS 상에서의 operation이 실제 operation으로 translate 될 수 있다.

## 구현

VFS는 리눅스에서 OOP로 구현된 subsystem 중 하나이다.
즉 Inode, superblock과 같은 객체 중심으로 구현되어 있다.
